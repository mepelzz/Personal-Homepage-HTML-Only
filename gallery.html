PROJECT 01

Desktop Application

DataBoost Inventori

Aplikasi desktop berbasis Java untuk manajemen inventori barang secara lengkap. Dibangun dengan arsitektur MVC dan terhubung ke database MySQL, aplikasi ini mendukung multi-role pengguna (Admin & Staf) dengan hak akses yang berbeda.

Java JavaFX / Swing MySQL MVC

Fitur Utama

▹ Dashboard monitoring stok dengan grafik real-time dan notifikasi stok kritis
▹ Modul transaksi penjualan dengan kalkulasi otomatis dan riwayat history
▹ CRUD data inventori lengkap dengan filter kategori dan pencarian barang
▹ Manajemen supplier dan ekspor data ke format CSV
▹ Multi-role: Admin memiliki akses penuh, Staf hanya dapat melihat dan bertransaksi
PROJECT 02
🔐

Machine Learning · Computer Vision

Gesture Lock

Sistem keamanan berbasis gestur tangan menggunakan kombinasi Rock Paper Scissors sebagai "password" visual. Model KNN dilatih menggunakan hand landmark dari MediaPipe untuk mengenali 3 kelas gesture secara real-time melalui kamera dengan akurasi 100% pada data uji.

Python MediaPipe OpenCV Scikit-learn KNN

Fitur Utama

▹ Deteksi 21 hand landmark secara real-time menggunakan MediaPipe
▹ Klasifikasi 3 gesture (Batu, Gunting, Kertas) dengan model KNN
▹ Sistem lock/unlock berdasarkan urutan kombinasi gestur sebagai password
▹ Akurasi model 100% pada data uji (Confusion Matrix KNN)
▹ Mendeteksi gesture UNKNOWN untuk input yang tidak dikenali
